== Description ==
This is a plugin that helps you visualize your Wordpress shortcodes.
With this plugin you can easily edit and preview how your shortcodes look on your website.

== Installation ==

Note: Plugin can be downloaded from the dreamForge Magic Shortcode Repository: https://github.com/dreamForge-forging-our-dreams-in-tech/Magic-Shortcode
Go to Plugins > Add Plugin > Upload Plugin > select Magic_Shortcode.zip > Install Now – succes: the plugin is installed!

== FAQ ==

= How do I use Magic Shortcode? =

The visualizer only works with it's own shortcode (Magic Shortcode) this means that any other shortcodes will just show up normally, if you use the plugins dedictated shortcode then you can preview any shortcode you select in the dropdown menu and modify it's attributes.

== Behind the scenes ==

= How does this plugin work? =

1. It scans all files installed in your wordpress directory for shortcodes and extracts all shortcodes and their attributes
2. The user adds the Magic Shortcode shortcode the their editor and it asks the user to select a shortcode they want to render.
3. The shortcode looks through its database of scanned shortcodes and retrieves the HTML and associated attributes.
4. It renders the HTML from the selected shortcode and displays the list of associated atttributes
